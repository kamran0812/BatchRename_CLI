
Usage: python rename.py [-h] --p [/path/..] --n [NAME] [--e .txt]

Example : python3 rename.py --p /dev/testfolder --n newName --e .txt    // This Will Rename files to "newName1.txt , newName2.txt ...."

Arguments:
  -h, --help      show this help message and exit
  --p [/path/..]  Path to files
  --n [NAME]      Enter Name of file
  --e .txt        Extension
